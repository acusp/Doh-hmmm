## 页面加载动画

> Ref: [How to Build a Delightful Loading Screen in 5 Minutes](https://medium.freecodecamp.org/how-to-build-a-delightful-loading-screen-in-5-minutes-847991da509f)

主要使用到的 CSS 技巧:

```
// 布局弹性布局
display: flex;

// 边框变为圆形
border-radius: 50%;

// css 动画
animation

// 创建动画关键帧
@keyframes
```
